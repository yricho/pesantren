===================================
CARA PAKAI:
===================================

1. Login Vercel Dashboard
2. Pilih Project kamu
3. Settings → Environment Variables
4. Add satu-satu:
   - Name: DATABASE_URL → Value: [copy dari atas]
   - Name: POSTGRES_URL → Value: [copy dari atas]
   - Name: PRISMA_DATABASE_URL → Value: [copy dari atas]
   - Name: NEXTAUTH_URL → Value: [copy dari atas]
   - Name: NEXTAUTH_SECRET → Value: [copy dari atas]
5. Save
6. Redeploy (Deployments → ... → Redeploy)
